# Soft Blue Gallery Planner

Личный веб-планер на Flask в стиле **Soft Blue Gallery**: кремовый фон, пастельно-голубая палитра, журнальная ретро-эстетика и календарь на три недели.

## Что уже есть

- Flask-приложение на Python.
- HTML/CSS/JavaScript-макет одной главной страницы.
- SQLite-база `planner.db`.
- Разовые задачи в таблице `tasks`.
- Регулярные задачи в таблице `regular_tasks`.
- Исключения для отдельных регулярных занятий в таблице `regular_task_exceptions`.
- Автоматический счётчик регулярных задач: `Английский · урок 4`.
- Выбор дня в календаре кликом.
- Листание дней в левой панели.
- Вертикальный скролл задач в панели “План на день”.
- Добавление, редактирование и удаление задач из левой панели и из календаря.
- Добавление регулярных задач через ту же форму.
- Редактирование регулярной задачи:
  - только выбранное занятие;
  - вся серия.
- Удаление регулярной задачи:
  - только выбранное занятие;
  - вся серия.

## Где что править

### `app.py`

Главный Flask-файл. Здесь:

- маршруты страницы;
- API для добавления, редактирования и удаления задач;
- сборка календаря на 3 недели;
- проверка данных из формы.

Основные API:

```text
GET    /api/calendar
GET    /api/day/<date>
POST   /api/tasks
PATCH  /api/tasks/<id>
PATCH  /api/tasks/<id>/done
DELETE /api/tasks/<id>
POST   /api/regular-tasks
PATCH  /api/regular-tasks/<id>
DELETE /api/regular-tasks/<id>
PATCH  /api/regular-occurrences/<id>/<date>
DELETE /api/regular-occurrences/<id>/<date>
```

### `database.py`

Работа с SQLite:

- создание таблиц;
- стартовые тестовые данные;
- получение задач на дату;
- создание/изменение/удаление задач;
- создание исключений для регулярных задач.

### `planner.db`

Файл SQLite-базы. Именно здесь сохраняются добавленные и изменённые задачи.

### `init_db.py`

Пересоздание базы.

Осторожно: команда ниже удалит текущие задачи и вернёт стартовые тестовые данные.

```powershell
python init_db.py --reset
```

### `templates/index.html`

Структура страницы:

- левая панель;
- календарь;
- модальное окно редактирования задачи;
- кнопки добавления/редактирования/удаления.

### `static/js/app.js`

Интерактивность:

- выбор дня;
- листание дней;
- открытие формы;
- отправка данных в Flask API;
- обновление календаря без перезагрузки страницы.

### `static/css/style.css`

Визуальная часть:

- стиль планера;
- карточки дней;
- кнопки при наведении;
- модальное окно;
- поля формы.

## Как пользоваться

### Добавить задачу в левой панели

Нажать кнопку `+ задача` рядом с заголовком “План на день”.

### Добавить задачу в календаре

Навести на нужный день и нажать `+ Добавить`.

### Редактировать задачу

Навести на задачу и нажать `✎`.

### Удалить задачу

Навести на задачу и нажать `×`.

### Сделать задачу регулярной

В форме выбрать тип `Регулярная задача`, затем указать:

- день недели;
- дату начала серии;
- подпись счётчика: `урок`, `занятие`, `тренировка` и т.д.
